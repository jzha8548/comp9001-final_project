Core Battle Logic

Turn order: the creature with higher SPD attacks first. If speed is tied, the player creature moves first.

Damage: max(1, ATK + skill power - DEF) × type multiplier. The multiplier comes from a two-dimensional type chart.

Expected skill score damage × accuracy The simulator chooses the skill with the highest expected score.
